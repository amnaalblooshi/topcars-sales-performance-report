# TopCars Sales Performance Report

## Overview
This project was developed as part of a Power BI training program.

The report analyzes sales performance for TopCars Co. Ltd through interactive dashboards and business intelligence techniques.

## Key Features
- Data cleaning and transformation
- Data modeling
- DAX calculations and measures
- KPI tracking
- Sales performance analysis
- Manufacturer analysis
- Salesperson analysis
- Interactive dashboard design

## Report Pages
1. Sales Summary
2. Manufacturer Analysis
3. Sales Performance Analysis
4. Salesperson Analysis

## Tools Used
- Microsoft Power BI
- Power Query
- DAX
- Data Modeling

## Project Type
Training and portfolio project.
